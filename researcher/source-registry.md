# Source Registry

## Research Priorities

1. **Attention mechanics** — New findings on context window behavior, attention patterns, lost-in-middle effects
2. **Multi-agent coordination** — Production benchmarks, coordination overhead measurements, token multipliers
3. **Evaluation methods** — LLM-as-judge reliability, bias mitigation techniques, calibration approaches
4. **Tool design** — Consolidation evidence, MCP patterns, tool selection accuracy studies
5. **Memory architectures** — Vector RAG vs. graph memory comparisons, hybrid approaches

## Source Categories

### Primary Sources (highest signal)
- Academic papers with reproducible experiments
- Production case studies with measured outcomes
- Official documentation from model providers

### Secondary Sources (supporting evidence)
- Blog posts from practitioners with data
- Conference talks with technical depth
- Open-source project documentation

### Tertiary Sources (context only)
- Opinion pieces and commentary
- Tutorials and how-to guides
- Community discussions
