# How Skills Built This

## 1. `context-fundamentals`
- **Attention Budget**: Instead of dumping thousands of markdown notes into context, notes are indexed. Only highly relevant nodes are disclosed progressively.

## 2. `memory-systems`
- **Graph-based Memory**: Links between notes are treated as graph edges, allowing the agent to traverse related topics.

## 3. `filesystem-context`
- **File-based Indexing**: Uses a simple, lightning-fast filesystem search mechanism to retrieve content dynamically.
